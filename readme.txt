Simplify the payment process on your WooCommerce store with the official GETO payment gateway integration.

== Description ==

The GETO Payment Gateway plugin provides a secure and seamless checkout experience on your WooCommerce store using GETO's powerful payment platform.

**Key Features:**

* Fast and secure online payments through GETO
* Webhook support for asynchronous payment confirmation
* Live and test mode configuration
* Fully integrated with WooCommerce

Visit [https://geto.app](https://geto.app) to learn more or sign up for a GETO account.

== Installation ==

1. Download the plugin ZIP file.
2. Go to your WordPress admin dashboard.
3. Navigate to **Plugins > Add New** and click **Upload Plugin**.
4. Choose the ZIP file and click **Install Now**.
5. After installation, click **Activate Plugin**.
6. Go to **WooCommerce > Settings > Payments > GETO** to configure your API keys and settings.

== Frequently Asked Questions ==

= Is GETO available in my country? =
Please check [https://geto.app](https://geto.app) for availability by region.

= What currencies are supported? =
GETO supports multiple currencies. You can configure your store currency via WooCommerce settings.

= How do I test payments? =
You can toggle between **Live Mode** and **Test Mode** in the plugin settings.
